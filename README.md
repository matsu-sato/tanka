# tanka
地図APIを使った旅×短歌アプリ

# ①課題番号-プロダクト名
- 地図で歌詠みアプリ

## ②課題内容（どんな作品か）
- MAPBoxの地図APIを使ったアプリ
- 地名にピンを指して、歌を追加できます。



## ③工夫した点・こだわった点
- mapboxの基本コードのコピペからスタートして、欲しい機能を追加しました。
- 地図の表示起点を地元（大阪府豊中市）に設定


## ⑥難しかった点・次回トライしたいこと(又は機能)
- 地図を日本語表記にしたいのにできない・・・（諦めた）
- マウスオンで表示がしたかった・・・（チャレンジしたけどできなかった）
